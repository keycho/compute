the execution layer for distributed compute

q0r routes workloads across a global mesh of independent nodes. placement resolves on live latency, stake weight, and available capacity. every execution is verified and finalized at the protocol layer.

01 · topology

a mesh that re-forms around demand

every node announces capacity, latency profile, and stake weight to the routing layer. hot regions attract flow. saturated nodes shed it. placement is not scheduled - it is an emergent property of the mesh.

latency

propagation is measured continuously between peers. routing reads the live latency field, not a static region map.

stake

operators bond stake to their nodes. weight scales placement priority and is slashed on failed verification.

capacity

nodes stream available compute envelopes. the mesh routes around saturation before queues can form.

02 · execution path

one job, end to end

01

submit

a job enters the network as a signed execution request: image digest, resource envelope, price ceiling, deadline. nothing in the request names a machine - the mesh decides.

execution.submit(request, signature)
02

route

the routing layer scores candidate nodes on live latency, stake weight, and free capacity. placement resolves in a single round. p50 is 38 ms from request to assignment.

score = f(latency, stake, capacity)
03

execute

the assigned node runs the workload in an attested sandbox. progress and resource draw stream back as signed telemetry, visible to the submitter in real time.

telemetry ← node · signed · 1 hz
04

verify

output commitments are checked against redundant execution or proof attestation, then finalized to the protocol log. settlement releases at finality. failed verification slashes stake.

finalize(commitment) → epoch log

04 · protocol surface

a small surface, precisely specified

q0r exposes five verbs. everything else is protocol behavior.

execution.submit

sign and broadcast an execution request. returns a job id at acceptance.

execution.status

stream placement, telemetry, and verification state for a job.

mesh.nodes

enumerate announced nodes with live latency, stake, and capacity signals.

mesh.topology

read the current mesh graph as weighted edges.

settlement.epoch

query accrual and finalized settlement for any epoch.
